# 🌾 KrishiAI — AI Powered Smart Farming Platform

KrishiAI is an AI-powered smart agriculture platform focused on helping farmers make intelligent farming decisions using Artificial Intelligence, weather analytics, crop intelligence, and disease detection systems.

The platform combines modern web technologies with AI/ML models to improve agricultural productivity and simplify smart farming.

---

# 🚀 Features

✅ AI-Based Crop Recommendation  
✅ Plant Disease Detection  
✅ Smart Farming Guidance  
✅ Weather Forecast Integration  
✅ Fertilizer Recommendation  
✅ Responsive Modern UI  
✅ AI Agriculture Insights

---

# 🧠 Technologies Used

- HTML5
- CSS3
- JavaScript
- Python
- Machine Learning
- Streamlit
- AI/ML Models

---

# 🎯 Project Vision

KrishiAI aims to empower farmers with intelligent digital farming tools that improve crop productivity, disease management, and agricultural decision-making using Artificial Intelligence.

### 🌦 Weather Forecast
